# Pizza Ordering App

An app for ordering and managing your pizzas.

## Installation

Use npm or yarn.

```bash
npm install
npm start
npm run ios
```

## Tools used

- [React native](https://reactnative.dev/)
- [styled components](https://styled-components.com/)
- [Firebase](https://firebase.google.com/)
